四大实际应用场景

APPLICATION SCENARIOS
AI 从 「临时顾问」 变成 「常驻伙伴」 · 从 「答题机器」 变成 「成长搭档」
场景一
长期项目管理
AI = 项目成员
TELOS 记录项目目标与方向
Memory 记住所有设计决策
/project-status 快速生成进度报告
Hooks 自动同步 Git 与 TODO
场景二
知识积累
AI = 学习伙伴
对话记录存入情景记忆
重要概念提炼到语义记忆
知识领域文件随学习持续更新
/review-learning 定期复习巩固
场景三
团队协作
AI = 团队基础设施
导出/导入脱敏版 TELOS 配置
Skills 在团队间共享复用
Memory 记录团队共识与规范
统一团队 AI 协作标准
场景四
创意工作
AI = 创意搭档
风格文件定义你的语言风格
价值观文件确保建议符合理念
Memory 记住创意演化过程
/brainstorm 基于历史产生新想法
8 / 10
