整个系统分为九个步骤，覆盖从文案策划、语音合成、字幕生成到最终视频渲染的完整链路。AI 的职责被简化为按照既定规则生成符合语法规范的 JSON 配置，而非直接编写代码。
核心
AI 不写代码
只生成 JSON 配置
将 AI 的输出限定为结构化的 JSON 数据，由确定性的模板引擎负责渲染，从根本上消除了 AI 生成代码的不可预测性。
可控性大幅提升
JSON 输出可被 Schema 严格校验，避免了 AI 生成代码时常见的语法错误、逻辑漏洞和运行时异常。

新增功能只需添加模板组件和对应的 JSON 规则，无需修改核心渲染引擎，实现真正的插件化架构。
+ 新模板 → + JSON 规则 → 新功能上线
