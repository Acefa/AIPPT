Hooks与Skills让AI主动感知并学会新能力

Hooks 事件钩子
给AI安装「传感器」，主动感知环境
核心理念
借鉴 Git Hooks 概念，在特定事件发生时自动触发，让AI从被动等待变为主动感知
EVENT TRIGGER FLOW
onPrompt
提问前
自动获取 Git 状态 + 当前任务
onResponse
回复后
自动记录时间戳 + 操作日志
onToolCall
调用工具前
先同步工作区状态 + 校验
全自动 · 零干预 · 实时感知
Skills 能力插件
可命名、参数化、可组合的AI扩展
SKILL COMMANDS
$ /review-pr --branch=feat/auth
$ /commit --conventional --scope=api
$ /analyze-logs --last=24h --level=error
$ /deploy --env=staging --dry-run
SKILL COMPOSITION
/full-review
复合工作流
/lint
/test
/review-pr
📦 社区共享
🔧 团队定制
🧩 类似 VS Code 扩展市场
Hooks 让AI感知环境，Skills 让AI学会新能力 —— 用自然语言「编程」AI的行为，这是真正的可编程AI
